### 创建用户
```sql
CREATE USER 用户名 IDENTIFIDE BY '密码'
```
### 授权
```sql
GRANT privileges ON 数据库.表 TO 用户名/public(所有人)
```
- `privileges`：
	- `update`
	- `select`
	- `insert`
	- `delete`
	- `all(update、select、insert、delete)`
### 撤销授权
==账户需要重新连接数据库才会生效==
```sql
REVOKE privileges ON 数据库.表 FROM 用户名
```
### 删除用户
==账户需要重新连接数据库才会生效==
```sql
DROP USER 用户名
```
