### 香农定理
$C=Wlog_2(1+S/N)$
C -> 信道容量`b/s`
W -> 信道带宽
N -> 噪声功率
S -> 信号功率
S/N -> 信噪比
- 信噪比越大，信道质量越好
- 一般用$10log_{10}(S/N)$表示信噪比，单位为分贝dB

### 传输介质
#### 双绞线
成本低，易于铺设
#### 同轴电缆
用得少
#### 光纤
1. 单模光纤
频带宽，传输距离长
2. 多模光纤
频带窄，传输距离短
- 重量轻
- 传输速率高
- 误码率低
- 传输损耗少，保密性好
- 成本高
