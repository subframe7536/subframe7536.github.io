#### 中断
- 在计算机执行期间，系统内发生任何非寻常的或非预期的急需处理事件使得CPU暂时中断当前正在执行的程序而，转去执行相应的时间处理程序。待处理完毕后又返回原来被中断处继续执行或调度新的进程执行的过程。
- 外部中断、trap、exception 他们三者的区别
#### 直接内存访问：Direct Memory Access (DMA)
- 允许外围设备和主内存之间直接传输它们的I/O数据，而不需要系统处理器的参与。
- 中断与DMA的区别
