### 动画过渡
给主属性加的属性，不是给:hover
```css
transition: 过渡的属性 花费的时间 运动曲线 何时开始;
```
过渡的属性：属性/all
运动曲线：

| 曲线        | 含义               |
| ----------- | ------------------ |
| easy        | 逐渐减速(默认)     |
| linear      | 匀速               |
| easy-in     | 加速               |
| easy-out    | 减速               |
| easy-in-out | 先加速后减速       |
| steps(n)    | 分步走(用于帧动画) |
何时开始：什么时候触发，默认为0
多个属性用`,`分隔
