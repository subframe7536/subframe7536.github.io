### JS执行队列
> 单线程，但利用多核cpu，创建多个线程，出现同步和异步

异步：多个任务同时执行(有回调函数)
同步任务：放在主线程执行栈，同时执行
异步任务：不同时进行(`click`/`resize`)、资源加载(`load`/`error`)、定时器(`setTimeout`/`setInterval`)
- 先执行执行队列的同步任务，将回调函数放到任务队列，跳过并继续执行下一条任务。同步任务执行完毕后，异步进程将触发条件的异步任务放到执行队列执行
