### 标题
`<h1>111</h1>`
h1~h6六级标题
### 段落
`<p></p>`
分段落
### 换行
`<br/>`  
单标签，强制换行
### 分割线
`<hr>`  
分隔元素
### 文字格式化标签
一般在css中写

| 格式   | 标签                           | 推荐(语义上更强调)  |
| ------ | ------------------------------ | ------------------- |
| 加粗   | `<strong></strong>`或`<b></b>` | `<strong></strong>` |
| 斜体   | `<em></em>`或`<i></i>`         | `<em></em>`         |
| 删除线 | `<del></del>`或`<s></s>`       | `<del></del>`       |
| 下划线 | `<ins></ins>`或`<u></u>`       | `<ins></ins>`       |

- `<i></i>`常用在字体图标中(较短)

### 注释 
`<!--111-->`
